# Краудфандингова Платформа на Django Повноцінна краудфандингова система з українською локалізацією, системою ролей, функціями пошуку, та медіа-підтримкою проектів. ## Функціональні можливості - Українська локалізація інтерфейсу - Система ролей (Інвестор, Автор, Адміністратор) - Реєстрація та автентифікація користувачів - Створення, редагування та управління проектами - Функції пошуку, сортування та фільтрації проектів - Система донатів з прогрес-баром - Медіа-підтримка проектів (зображення, відео, документи) - Чат між авторами та інвесторами - Адміністративна панель для управління користувачами та проектами ## Технічні особливості - Django 4.0+ - Bootstrap 5 - Система шаблонів та CSS для адаптивного дизайну - Паралакс-ефект для фонових зображень - Кольорова гама, розроблена на основі природних кольорів - SQLite база даних - Багаторівнева система безпеки ## Встановлення 1. Клонуйте репозиторій: git clone https://github.com/yourusername/crowdfunding.git cd crowdfunding 2. Створіть віртуальне середовище та активуйте його: python -m venv venv source venv/bin/activate # Linux/Mac venv\Scripts\activate # Windows 3. Встановіть залежності: pip install -r requirements.txt 4. Виконайте міграції: python manage.py makemigrations python manage.py migrate 5. Створіть суперкористувача: python manage.py createsuperuser 6. Запустіть сервер розробки: python manage.py runserver 7. Компіляція файлів перекладу: python manage.py compilemessages ## Вимоги - Python 3.8+ - Django 4.0+ - Pillow (для обробки зображень) - Django-crispy-forms ## Ліцензія Цей проект розповсюджується під ліцензією MIT.
